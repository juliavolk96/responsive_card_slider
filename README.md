# Card Slider

A sleek, responsive character slider built with **Swiper.js**, featuring a custom "inverted curve" CSS overlay effect.

## Features
- **Responsive**: Mobile-first design (PC, Tablet, Mobile support).
- **Custom UI**: Advanced CSS pseudo-element tricks for card overlays.
- **BEM Design**: Maintainable and clean code structure.

## Tech Stack
- **HTML5** (Semantic)
- **CSS3** (Flexbox, Variables, Pseudo-elements)
- **JavaScript** (ES6+)
- **Swiper.js** (Slider Engine)

## Structure
- `index.html` - Semantic markup
- `css/style.css` - Custom BEM styles
- `js/script.js` - Swiper configuration
- `images/` - Character portraits

## Quick Start
1. Clone the repo.
2. Open `index.html` in your browser.
